# Planning Examples

## Good Snippet

```markdown
## Skill Routing

- planning kickoff: `project-planning`
- implementation orchestration: `big-task`
- contract safety: `api-contract-guard`
- document closure: `doc-sync`
```

이 예시는 "언제 어떤 스킬을 쓰는가"가 바로 보인다.

## Bad Snippet

```markdown
## Skill Routing

- use some planning skills later
```

이 예시는 실제 호출 시점과 스킬명이 없어 실행 계약으로 쓸 수 없다.
