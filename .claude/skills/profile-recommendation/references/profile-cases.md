# Profile Cases

## High Confidence

- `apps/web`, `backend/app`, Supabase env token, web verify command -> `Next.js + FastAPI + Supabase`
- `backend/`, `workers/`, python verify command -> `Python API + Worker`

## Low Confidence

- frontend와 worker가 같이 있고 backend 경계가 얕은 경우
- verify command가 문서 검사뿐이라 runtime signal이 약한 경우
