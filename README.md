# UR HYNIX Project Docs

UR HYNIX 팀의 TurtleBot3 + Unity Digital Twin + Camera Vision 장애물 인식 프로젝트 로컬 문서 보관소입니다.

이 폴더는 Jira/Confluence에 올린 내용을 다시 찾기 쉽게 정리하는 로컬 정본입니다.

8. [결정 기록](docs/status/DECISION-LOG.md)

## 문서 운영 규칙

- Jira는 작업 관리용, 이 폴더는 판단 근거와 발표 자료 준비용으로 사용합니다.
- 큰 결정이 바뀌면 `docs/status/DECISION-LOG.md`에 남깁니다.
- 개발 진행 상황은 `docs/status/PROJECT-STATUS.md`에 짧게 갱신합니다.
- 실제 테스트 사진, 영상, 로그는 `docs/evidence/` 아래에 날짜별로 둡니다.
- 발표에 쓸 이미지나 화면 캡처는 `docs/assets/` 아래에 둡니다.
