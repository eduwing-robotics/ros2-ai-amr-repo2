# Daily Notes

> 일일 스크래치 노트 폴더. 세션 중 임시 작업 기록.

## 용도

- 한 세션 내 빠른 메모, 스크래치, 임시 의사결정
- 완성 후 정식 문서(evidence/, status/, ref/)로 이관

## 보관 규칙

- **최신**: MMDD.md (오늘 날짜)
- **완성 후**: evidence/ 또는 status/로 이관
- **미수정 3개월 이상**: 분기별 아카이브 폴더로 자동 정리

## 예시 경로

```
docs/
├── daily/
│   ├── 0617.md        (오늘)
│   └── 0616.md        (어제)
├── archive/
│   ├── daily-2026-05/  (5월 미수정 아카이브)
│   │   ├── 0526.md
│   │   └── 05-27-recap-pending.md
│   └── daily-2026-04/  (4월 아카이브)
├── evidence/          (완성된 실험/검증 기록)
└── status/            (프로젝트 상태, 의사결정)
```
