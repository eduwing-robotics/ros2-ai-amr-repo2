# Technology Ref Folder

이 폴더는 기술별 빠른 진입점이다. 정본 내용을 길게 복제하지 않고, 작업자가 특정 기술 작업을 시작할 때 읽을 최소 문서와 검증 명령만 연결한다.

## Rules

1. 새 기술 ref는 `docs/ref/TECH-INDEX.md`에 라우팅 표를 먼저 추가한다.
2. 각 기술 ref는 1페이지 안쪽을 목표로 한다.
3. 세부 설계, 긴 절차, evidence는 기존 정본 문서나 `.claude/skills/*/references`에 둔다.
4. 기술 ref를 수정하면 관련 스킬 README, `PROJECT-PLAN.md`, `PROJECT-STATUS.md` 중 필요한 문서와 동기화한다.
